### Making a background element

There's a few ways to create the background element. An 'easy' but hard to understand solution is with the `border-image` property. But that one doesn't support `border-radius` which you often need.

Let's start by laying out a new element underneath the element where we want the border.

```css
.element-with-gradient-border:after {
  /* we need 'some' content, otherwise the button won't show */
  content: '';
  /* Absolute position, to position the background underneath the button */
  position: absolute;
  /* Gradient should hide underneath the main element */
  z-index: -1;
  /* Use inset to make element fill the same area as its parent
     You cannot just use height: 100%; width: 100%; in this case 
     because the element would be pushed around by adding padding for example
  */
  inset: 0;

  /* We can then simply use a negative margin to control how thick the border should be */
  margin: -2px;
}
```
